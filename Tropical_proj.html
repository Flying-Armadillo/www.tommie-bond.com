Virtual Photorealistic Tropical Island

This project was an exploration the tools and techniques used in the creation of virtual photorealistic environments in Unreal engine. A tropical island theme was chosen as I was inspired by a day trip I had to Tangalooma resort on Morton Island in March of 2023. This also had the added benefit of being able to implement some of the experimental water system plugin to create an ocean.
The tools used for this project was Unreal engine, World machine basic, and Adobe Premiere Pro

World machine basic generated the height map used to shape landscape to create the islands.
The water systems plugin was added and used to create the ocean.
High quality textures and models were downloaded from the Quixel Bridge in Unreal.
The textures were blended using 'landscape layer blend' and 'linear interpolation'.
The Procedural Foliage Tool was used to place the plants, rocks, and coral randomly across the island.
Camera Rails and Camera Actors were used to set up the shots.
Adobe Premiere Pro was used to combine the renders together, edit the audio, and render the final video.
